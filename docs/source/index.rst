HPC Cluster at INFN Padova User Guide
=====================================
This guide describes how the HPC Cluster at INFN Padova is configured and can be
used.

It is not intended to be a SLURM user guide: for that please refer to the `SLURM
official documentation <https://slurm.schedmd.com/>`__.

.. note::

   This documentation is under development.
